# Baião Society - Frontend

Interface React para o sistema de gestão de quadras.

## Setup Rápido

```bash
npm install
npm start
```

**Acesso:** http://localhost:3000

## Estrutura

```
src/
├── components/    # Componentes React
├── pages/         # Páginas principais
├── services/      # Serviços API
├── hooks/         # Custom hooks
├── utils/         # Utilitários
└── context/       # Context API
```

## Variáveis de Ambiente

Copiar `.env.example` para `.env`.

## Testes

```bash
npm test
```

## Build

```bash
npm run build
```

Deploy em Vercel:
```bash
vercel deploy
```
